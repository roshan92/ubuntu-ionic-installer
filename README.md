Ubuntu Ionic Installer
======================

Installs Android, Apache Ant, Java JDK, NodeJS, Apache Cordova, and Ionic Framework on Ubuntu x64 and x86 architectures.  Has been tested with Ubuntu 14.04 Trusty Tahr.
